у программы есть gui

на вход должен подаваться:

сайт для парсинга
название марки
название модели
год

на выходе 4 картинки на листе а4
на каждой эмблема (эмблемы хранятся в папке проекта)
страна производства (флаги хранятся в папке проекта)
мощность в лс
максимальная скорость в км/ч

через gui возможность отправки на печать
сохранение в jpg всего листа
кнопка next с другими вариантами (все 4)

на каждой из 4 частей - возможность замены картинки
возможность редактирования текста вручную

база:
марка - файл с флагом - файл с эмблемой

при DPI = 150, разрешение формата А4 имеет 1240 × 1754 пикселей;
при DPI = 300, разрешение формата А4 имеет 2480 × 3508 пикселей.
